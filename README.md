# KITools
KITools is a website with some online tools that can come in handy.

## Demo
https://kira-iovenko.github.io/KiTools/

## Feautures:
- Whitespace Remover
- HTML Preview
- JSON Formatter
- Timer
- Pomodoro Timer
- Coin Flip
- Flashcards
- Todo List
- QR Code Generator
- Banner Generator

## Queenie pet
A built-in interactive companion!

- Appears on every page
- Sprite animations
- Can be toggled to hide/show

## Dark Mode
KITools supports dark mode!

-Toggle for switching between light/dark themes
